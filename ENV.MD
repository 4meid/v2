# Ví Dụ Cụ Thể

# Bước 1: Tạo môi trường ảo
```
python3.10 -m venv myenv
```
# Bước 2: Kích hoạt môi trường ảo
# Trên Windows
```
.\myenv\Scripts\activate
```
# Trên macOS và Linux
```
source myenv/bin/activate
```
# Bước 3: Cài đặt gói
```
pip install requests
```
# Bước 4: Tắt môi trường ảo
```
deactivate
```
